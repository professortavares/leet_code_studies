# Leet Code Studies

Este projeto reúne soluções didáticas para problemas de algoritmos, escritas em notebooks Jupyter e organizadas para leitura em formato de livro.

A proposta é estudar cada problema com calma, comparar abordagens diferentes e construir um material que possa evoluir com o tempo. Conforme novos notebooks forem adicionados, o livro pode crescer de forma natural, mantendo a navegação simples e objetiva.

## O que você encontra aqui

- explicações em português do Brasil
- soluções comentadas passo a passo
- análise de complexidade
- testes simples e confiáveis
- material pronto para ser convertido em um livro com Jupyter Book

## Como navegar

Use o menu lateral para acessar os problemas resolvidos. A primeira seção do livro já está organizada para receber novos notebooks conforme o repositório for crescendo.
